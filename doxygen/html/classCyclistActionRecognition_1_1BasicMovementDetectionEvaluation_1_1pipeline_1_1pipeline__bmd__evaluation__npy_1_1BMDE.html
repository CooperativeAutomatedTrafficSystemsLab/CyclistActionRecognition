Detailed Description

Basic Movement detection evaluation.

This class contains methods to evaluate basic movement detections. The generated scores and plots are F1-scores, seg-scores, confusion matrix, qq plots, brier scores, and detection times.

It is also possible to create plots for every scene containing the trajectory showing the ground truth classes for every position and a plot containing the predicted classes compared to the ground truth.

Parameters
npy_path(str): path to npy file containing test results in the form of a dict:
1 {SCENE_NAME_0: {"x_tracked": X_POSITIONS,
2  "y_tracked": Y_POSITIONS",
3  "labels_BM0": BM0 LABELS (0 or 1),
4  "pred_BM0: BM0 PREDICTIONS (between 0.0 and 1.0),
5  ...
6  "pred_BMn: BMn PREDICTIONS},
7  SCENE_NAME_1: {...}, ...}
class_names(list(str)): list of basic movement names
output_path(str): where to save outputs, if None: npy_path is used
time_step(float): size of one time step in seconds
create_scene_outputs(bool): whether or not to create scene wise output (might take some time)
input_length(int): if trajectory is longer than predictions, the first input_length steps are skipped (only necessary when create_scene_outputs is set to True)
dont_use(list(str)): list of scenes names to exclude from evaluation
